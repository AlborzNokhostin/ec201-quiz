# EC201 International Macroeconomics Quiz

An interactive MCQ quiz covering Weeks 1-3 of EC201 International Macroeconomics, featuring a leaderboard system to track high scores.

## Features

- **35 comprehensive questions** covering:
  - Week 1: Global Imbalances
  - Week 2: Trade Balance and Current Account Sustainability
  - Week 3: Theory of the Current Account

- **Interactive learning experience**:
  - Immediate feedback on answers
  - Detailed explanations for each question
  - Progress tracking
  - Week-specific or combined quizzes

- **Leaderboard system**:
  - Enter your name before starting
  - Scores saved to browser localStorage
  - Top 10 scores displayed with rankings
  - Gold, silver, and bronze badges for top 3

## How to Use

1. Enter your name
2. Select which week(s) to be tested on
3. Click "Start Quiz"
4. Answer questions and receive instant feedback
5. View your final score and leaderboard ranking

## Topics Covered

### Week 1: Global Imbalances
- Balance of Payments components
- Net International Investment Position (NIIP)
- Valuation changes
- NIIP-NII paradox
- Dark Matter and Return Differential hypotheses

### Week 2: Sustainability
- Finite vs infinite horizon models
- Trade balance sustainability
- Transversality conditions
- Perpetual current account deficits
- Debt-servicing policies

### Week 3: Theory of Current Account
- Small open economy model
- Euler equations and optimal consumption
- Temporary vs permanent shocks
- Consumption smoothing
- Anticipated income shocks

## Technical Details

- Built with vanilla HTML, CSS, and JavaScript
- No external dependencies
- Works offline (except for hosting)
- Responsive design for mobile and desktop
- Data persists using browser localStorage
